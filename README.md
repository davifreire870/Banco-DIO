# Desafio POO - Sistema Bancario

Projeto desenvolvido durante os estudos de Python na DIO, com foco na evolucao de um sistema bancario usando conceitos de Programacao Orientada a Objetos.

## Objetivo

Criar uma aplicacao de terminal para simular operacoes bancarias simples, organizando as responsabilidades do sistema em funcoes e classes.

## Funcionalidades

- Criar usuarios
- Criar contas bancarias
- Listar contas cadastradas
- Realizar depositos
- Realizar saques
- Exibir extrato
- Controlar limite de saque
- Controlar quantidade maxima de saques

## Estrutura dos arquivos

```text
POO/desafio/
├── main.py
├── funcoes.py
├── desafio.ipynb
└── README.md
```

## Arquivos principais

- `main.py`: arquivo principal da aplicacao, responsavel pelo menu e pelo fluxo de execucao.
- `funcoes.py`: contem as classes criadas para representar clientes, contas, historico e transacoes.
- `desafio.ipynb`: notebook usado para estudo, testes ou desenvolvimento do desafio.

## Conceitos praticados

- Classes e objetos
- Heranca
- Encapsulamento
- Propriedades com `@property`
- Metodos de classe com `@classmethod`
- Classes abstratas com `ABC`
- Organizacao de codigo em modulos
- Funcoes com argumentos posicionais e nomeados

## Como executar

No terminal, acesse a pasta do desafio, dpois execute:

```bash
python main.py
```

Se estiver usando Linux ou macOS e o comando `python` nao funcionar, tente:

```bash
python3 main.py
```
